# Enterprise-MCP-AutoGen

AI Personal Operating System (AI-POS) - An enterprise-grade Agentic AI platform that acts as a user's digital chief of staff.

## Features
- **Frontend**: Next.js 15, TailwindCSS, Shadcn UI
- **Backend**: FastAPI, PostgreSQL + pgvector
- **AI Agents**: AutoGen with Groq (Llama 3)
- **MCP Integration**: Notion, GitHub, Gmail, Calendar
